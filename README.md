# Customer Churn Analysis

## Project Overview
This project focuses on analyzing telecom customer churn data using SQL and Power BI. The goal of the project is to identify the major factors affecting customer churn and provide business insights to improve customer retention.

---

## Dataset
Dataset used for this project:

- Telco Customer Churn Dataset from Kaggle  
- [Kaggle Dataset Link](https://www.kaggle.com/datasets/ylchang/telco-customer-churn-1113)

---

## Tools & Technologies Used
- Microsoft Excel
- MySQL
- Power BI

---

## Project Workflow
1. Data Cleaning using Excel
2. Data Import into MySQL
3. SQL Analysis and Query Execution
4. KPI Creation
5. Power BI Dashboard Development
6. Business Insights and Recommendations

- Churn customer analysis

## Dashboard Features
- KPI Cards
- Churn Distribution Pie Chart
- Contract Type Analysis

---

## Key Insights
- Month-to-month contract customers showed the highest churn.
- Customers with higher monthly charges were more likely to leave.
- Electronic check users had higher churn rates.
- Long-term contract customers showed better retention.

---

## Recommendations
- Introduce better retention offers for high-risk customers.
- Encourage long-term subscription plans.
- Improve customer support and engagement.
- Provide loyalty benefits for existing customers.
